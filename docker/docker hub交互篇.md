# docker 命令说明

## docker hub篇
```
1.  注册docker hub账号

2.  本地登录docker hub：docker login -u username -p password
    username：docker用户名
    password：docker密码

3.  本地退出docker hub：docker logout

4.  本地镜像重命名：docker tag xxx 账号/xxx
    xxx: 镜像名

5.  上传镜像到docker hub：docker push 账号/xxx
    xxx: 镜像名

6.  浏览器登录docker hub查看已上传的镜像
```

## 注意
```
1.  本地预先生成docker镜像文件
```
